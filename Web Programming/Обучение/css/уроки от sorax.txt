3 урок:

h1,li{} - чтобы дать правило для нескольких элементов, то можно их перечислить через запятую.

h1.main{} или h1#main{}- чтобы дать правило только для элементов у которых есть id ="main" или class="main"

*{} - селектор который выделяет все элементы на странице.

ul *{} - выделяет все элементы в ul.

div h1{} - выделяем элементы h1 в div'e

.list>li - комбинатор прямого наследования, то есть будут выделяться элементы только этого типа только на этом уровне

h4+li{} первый элемент li которые идёт после h4

h4~li{} выделяет все элементы li идущие после h4

:link{} - все ссылки, которые не были нажаты
:visited{} - все ссылки, которые были нажаты

:hover - при наведение
:active - ссылка, которая в данный момент активна
:focus - для полей ввода

li:nth-child(2n){} - каждый второй элемент li будет выделяться. (2n) можно заменить на любое выражение, или odd - нечетные, even - четные.

li:nth-last-child(2d){} -  тоже самое, только нумерация с конца списка

li:first-child{} - первый элемент в списке
li:last-child{} - последний элемент в списке

p::before{} - добовляет что-нибудь перед <p>
на пример:
p::before{
content: "Кстати";
}
p::after{} - добовляет что-то после <p>

a[title] - все <a> с атрибутом title

a[title="my-title"] - все <a> с атрибутом title с значением my-title

a[title~="my-title"] - одно из значений являеться указанным значением.

a[title|="my-title"] - <a> у которых title начинаеться с этого значения, а после него либо ничего, либо еще одно значение.

a[href^="http"] - будут выделены те ссылки у которых атрибут href начинаеться с этой строки

a[title$=".com"] - <a> у которых title заканчиваеться этим значением

a[title*="my-title"] - <a> у которых title встречаеться указанное значение.
